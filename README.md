# steo

Direction 

python > 3.7

1. Clone the repository to your local directory
`git clone https://github.com/JingweiZhang2017/steo.git` 

2. Change directory to 
`cd ../steo`

3. Set up the environment
`pip install -r requirements.txt`

4. Run the script
`python3 STEO.py`
